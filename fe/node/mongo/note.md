# MongoDB 数据持久化存储
- 是什么？
  - 1：一个基于文件存储的分布式 NoSQL 数据库系统
  - 2：数据结构由键值对(key, value)组成
  - 3：拥有非常强大的查询能力
- 特性？
  - 1：文档型数据库，较强可扩展性，拥有强大的查询语言，多种存储引擎
  - 2：高性能、高可用、水平扩展：支持数据嵌入，子文档查询、支持副本集与分片
  - 3：多种查询类型支持，且支持数据聚合查询、文本检索、地址位置查询
- 使用场景？
  - 1：对数据处理性能由较高要求
  - 2：需要借助缓存层来处理数据
  - 3：需要高度的伸缩性
