# 章介绍

JS 的特色就是异步编程，所有有很多关于异步的考点，本章都会讲解。如 event loop、promise、async-await、微任务和宏任务。学不会这些，就不算是精通 JS ，也无法进大厂。

## 产出

- 全面使用 Promise
- 学会 async await 语法
- 学会 event loop 流程
- 认识微任务和宏任务

## 主要内容

- event loop 执行过程
- Promise 全面使用
- async await 全面使用
- 微任务和宏任务
- 面试题讲解

## 关键字

- Promise
- async await
- event loop
- 微任务和宏任务
