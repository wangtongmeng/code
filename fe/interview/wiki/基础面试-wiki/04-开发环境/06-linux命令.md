# linux 服务器的基本命令

线上服务器一般使用 linux 服务器，而且是 server 版本的 linux，没有桌面也没有鼠标，如何操作呢？

**登录**

入职之后，一般会有现有的用户名和密码，拿来之后直接登录就行。运行 `ssh name@server` 然后输入密码即可登录

**目录操作**

- 创建目录 `mkdir`
- 删除目录 `rm -rf`
- 定位目录 `cd `
- 查看目录文件 `ls` `ll`
- 修改目录名 `mv `
- 拷贝目录 `cp`

**文件操作**

- 创建文件 `touch ` `vi `
- 删除文件 `rm`
- 修改文件名 `mv`
- 拷贝文件 `cp` `scp`

**文件内容操作**

- 查看文件 `cat` `head` `tail`
- 编辑文件内容 `vi `
- 查找文件内容 `grep `
